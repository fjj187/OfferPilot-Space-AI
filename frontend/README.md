# Frontend（Vue 3 前端）

宇宙模拟面试页与 Chat MVP 前端，基于 Vue 3 + Vite + TypeScript。

## 常用命令

在**仓库根目录**执行（推荐）：

```bash
pnpm dev              # 启动前端（等价于 dev:frontend）
pnpm dev:backend      # 启动后端 API
pnpm build
pnpm test
```

在本目录执行：

```bash
cp .env.template .env
pnpm dev
```

## 目录

- `src/` — 源码（页面、组件、composables、services）
- `__tests__/` — 单测
- `public/` — 静态资源

详见 [`docs/目录说明.md`](../docs/目录说明.md)。
