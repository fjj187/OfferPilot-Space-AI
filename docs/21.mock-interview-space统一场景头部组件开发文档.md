# 21.mock-interview-space（模拟面试星球页）统一场景头部组件开发文档

## 1. 背景

`/showcase/mock-interview-space`（模拟面试星球页）在第 20 阶段已经完成视觉一致性收尾方向确认，重点从功能扩展转为标题区、内容区、切换动效和滚动行为稳定。

本阶段目标是落地 `SpaceSceneHeader.vue`（宇宙场景头部组件），让五个场景在标题、说明和右侧辅助信息布局上使用同一套结构，减少后续新增场景时反复补高度、调间距的问题。

## 2. 本阶段目标

1. 新增 `SpaceSceneHeader.vue`（宇宙场景头部组件）。
2. 统一 `title`（标题）、`body`（说明）、`aside`（右侧插槽）三类头部内容入口。
3. 让各场景组件自己持有头部结构，避免由外层路由组件代替场景渲染标题。
4. 保持 `SpaceContentPanel.vue`（宇宙内容面板组件）只负责场景挂载、事件转发和滚动定位引用。
5. 保持 `SpaceGeneralScenePanel.vue`（通用场景面板组件）只负责总览、资料库等通用场景承接和右侧预览，不再承担统一头部渲染职责。

## 3. 已完成内容

### 3.1 统一头部组件

已新增 `SpaceSceneHeader.vue`（宇宙场景头部组件），当前支持：

1. `title`（标题）。
2. `body`（说明）。
3. `aside`（右侧插槽）。
4. `asideMinWidth`（右侧区域最小宽度配置）。

### 3.2 场景接入状态

已接入统一头部的场景：

1. `SpaceMockScene.vue`（模拟面试场景组件）。
2. `SpacePracticeScene.vue`（专项刷题场景组件）。
3. `SpaceReportScene.vue`（复盘报告场景组件）。
4. `SpaceLibraryScene.vue`（资料库场景组件）。

其中 `SpaceLibraryScene.vue`（资料库场景组件）已经从外层头部改为内部直接使用 `SpaceSceneHeader.vue`（宇宙场景头部组件），并去掉自身顶部多余 `margin-top`（顶部外边距），避免资料页标题比其他场景偏低。

### 3.3 不纳入本阶段的内容

`SpaceMockScene.vue`（模拟面试场景组件）的 `visibleMetaChips`（可见元信息胶囊计算属性）暂不做“最多 8 个并用 `+N`（剩余数量）收口”的处理。

## 4. 本轮继续开发内容

本轮继续收尾 `overview`（总览场景）的头部归属：

1. 让 `SpaceOverviewScene.vue`（总览场景组件）直接引入并使用 `SpaceSceneHeader.vue`（宇宙场景头部组件）。
2. 给 `SpaceOverviewScene.vue`（总览场景组件）补充 `sectionTitle`（场景标题）和 `sectionBody`（场景说明）两个 `props`（组件属性）。
3. 从 `SpaceGeneralScenePanel.vue`（通用场景面板组件）移除外层 `SpaceSceneHeader`（宇宙场景头部组件）渲染。
4. 确保 `overview`（总览场景）和 `library`（资料库场景）的标题高度由各自场景组件内部控制。

## 5. 职责边界

### 5.1 `SpaceSceneHeader.vue`（宇宙场景头部组件）

负责：

1. 统一标题字号、行高、说明文字间距。
2. 提供右侧 `aside`（右侧插槽）布局。
3. 响应式收拢右侧区域。

不负责：

1. 单个场景业务卡片布局。
2. 单个场景数据组装。
3. 页面滚动和轨道切换。

### 5.2 `SpaceGeneralScenePanel.vue`（通用场景面板组件）

负责：

1. 挂载 `SpaceOverviewScene.vue`（总览场景组件）。
2. 挂载 `SpaceLibraryScene.vue`（资料库场景组件）。
3. 资料库右侧 `DocumentPreviewPanel`（文档预览面板）渲染。
4. 通用右侧说明卡片渲染。

不负责：

1. 具体场景头部结构。
2. 具体场景标题高度。
3. 具体场景内部业务布局。

## 6. 验收清单

1. 五个场景标题均使用 `SpaceSceneHeader.vue`（宇宙场景头部组件）。
2. `SpaceGeneralScenePanel.vue`（通用场景面板组件）不再直接渲染场景头部。
3. 资料库标题不再比其他场景偏低。
4. 总览、资料库、模拟面试、专项刷题、复盘报告切换时标题区高度稳定。
5. `npm exec eslint -- src/components/showcase/mock-interview-space/SpaceGeneralScenePanel.vue src/components/showcase/mock-interview-space/scenes/SpaceOverviewScene.vue src/components/showcase/mock-interview-space/scenes/SpaceLibraryScene.vue`（执行相关文件代码规范检查）通过。
6. `npm run build`（执行生产构建验证）通过。
