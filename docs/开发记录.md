- **关键结论**：
  - 对外主链路仍以 `POST /api/interview/stream`（模拟面试流式接口）为核心，返回 `SSE`（服务端发送事件）流。
  - 推送事件名固定为 `chunk` / `done` / `error`，其中 `chunk` 事件 JSON（结构化数据）字段固定为 `content`。
  - 当前系统无鉴权、无应用层心跳、无自动重连；前端支持 `AbortController`（中止控制器）主动取消流。
